What is Git?
Git is a VCS(Version control system)that is used to store and share files in repositories. Git is also know as a 'distributed revision control system' which means it focuses on speed and keeping the integrity of the data intact.
Who started Git>
The credit for creation of Git is gifted to 'Linus Korvalds', which he created in 2005, but since that year 'Junio Hamano has been maintaining it.

What is the 'Repository' and what is its role?
A repository is a collection of files. Like the WINSCP application, Github holds files and broadcasts them from a central server.
